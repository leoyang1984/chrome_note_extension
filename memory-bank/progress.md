## 部署和配置指南

### 当前部署要求
1. **Logseq配置**:
   - 启用HTTP服务器（端口12316）
   - 设置API token为: `Abc123!`
   - 确保Logseq桌面应用运行

2. **浏览器扩展**:
   - 加载修改后的扩展
   - 确保有访问localhost权限
   - 网络连接正常

### 配置步骤
```bash
# 1. 在Logseq中配置
设置 → 高级 → HTTP服务器 → 启用
设置 → 高级 → API Token → 设置为 "Abc123!"

# 2. 在Chrome中加载扩展
chrome://extensions → 加载已解压的扩展程序 → 选择项目目录

# 3. 测试功能
打开任意网页 → 点击扩展图标 → 编写笔记 → 点击"📝"保存到Logseq
```
